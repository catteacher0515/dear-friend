# 生日祝福网站设计文档

## 项目概述

一个单人使用的生日祝福静态网站，包含信封拆信交互和时间线两个核心模块。内容通过配置文件管理，部署在 Vercel 上。

## 交互流程

```
场景一：信封桌面
- 桌面场景背景，信封居中偏下，占屏幕 60-70%
- 提示文字引导用户点击（如"点击打开"）
- 背景装饰元素体现生日氛围（后期填充）

↓ 用户点击信封

拆信动画（仍在场景一）
- 信封盖子翻开
- 信纸从信封内缓缓抽出
- 信纸展开，文字逐渐显示

场景一：阅读信件
- 信纸展开后显示完整信件正文
- 底部有"继续"按钮或向下滑动提示

↓ 用户点击继续

场景二：时间线
- 页面过渡动画切换
- 时间线从上到下按时间顺序排列
- 每条记录包含：时间、标题、文字描述、可选多张图片
```

## 数据结构

内容存储在 `public/config.json`，运行时 fetch 读取，修改内容无需重新构建。

```json
{
  "letter": {
    "recipient": "小明",
    "content": "亲爱的小明，..."
  },
  "timeline": [
    {
      "date": "2020-06-01",
      "title": "我们第一次见面",
      "description": "那天天气很好...",
      "images": ["images/01.jpg", "images/02.jpg"]
    },
    {
      "date": "2021-03-15",
      "title": "一起去了海边",
      "description": "...",
      "images": []
    }
  ]
}
```

- `letter`：信件收件人和正文
- `timeline`：数组，条目数量不限
- `images`：空数组表示不配图，支持多张图片（轮播或并排展示）

## 技术架构

- **框架**：Vue 3 + Vite
- **动画**：Vue `<Transition>` 组件 + CSS 动画
- **路由**：无需 vue-router，场景切换通过组件状态管理
- **图片轮播**：Swiper.js
- **部署**：Vercel，推送代码自动更新

## 目录结构

```
src/
  components/
    EnvelopeScene.vue   # 信封桌面场景（含拆信动画）
    LetterScene.vue     # 信纸阅读场景
    TimelineScene.vue   # 时间线场景
    TimelineItem.vue    # 单条时间线记录
  App.vue               # 场景状态管理与切换
public/
  config.json           # 内容配置
  images/               # 图片资源
```
